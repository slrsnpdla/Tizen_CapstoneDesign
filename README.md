- GPS 없이 Map Matching 기법을 통해 길안내를 하고자 한 어플리케이션입니다.
- 휴대폰 디스플레이를 지속적으로 키지 않고 보조적인 타이젠 워치를 사용해서 길 안내를 글자로 받는 방식입니다.

- 시연시 소스코드가 일부 분실되어 과거 소스코드를 통해 복원해서, 약간 버그가 있거나 일부 미구현상태입니다.

- 현재 사용시 필요한 권한
	* 저장장소(Map DB를 sdcard 최상단에서 불러옴)
	* 위치(GPS) (길안내에서는 쓰이지 않지만, Without GPS와 비교하기위해 로그기능으로 사용하긴 했습니다. 제거가능)
	* 마시멜로우 버전 이상의 안드로이드폰에서는 어플리케이션관리자 - 해당 어플리케이션 - 권한 에서 권한을 주어야합니다.

- 출발지와 도착지를 설정하고 검색하면, 벨만포드 알고리즘으로 그래프에서 최단경로를 구하고, 센서를 통해 이동한 거리를 측정합니다.
